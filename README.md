# markdown-to-html
Markdown to HTML converter

## Syntax

* `**Bold Text**` - **Bold Text**
* `__Bold Text__` - **Bold Text**
* `*Italic Text*` - *Italic Text*
* `_Italic Text_` - *Italic Text*
* `***Important Text***` - ***Important Text***
* `___Important Text___` - ***Important Text***
